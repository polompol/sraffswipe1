# StaffSwipe — one-pager

**Tinder для срочных смен в общепите.** Telegram Mini App: заведения за минуту
находят замену на смену, соискатели — подработку рядом с домом. Свайп → мэтч →
чат → подтверждение смены → акт для самозанятого.

## Проблема
Кафе/бары/рестораны постоянно «горят» из-за невыходов: официант заболел —
смену некем закрыть. Классические job-сайты медленные и не заточены под
«сегодня на вечер». Линейный персонал ищет подработку рядом и быстро.

## Решение
- **Свайп-лента смен** с гео-радиусом и фильтрами (роль/ставка/дата).
- **Мэтч → чат → подтверждение** смены в один тап; **PDF-акт** для самозанятого.
- **Вход через Telegram** (без SMS), уведомления ботом, оплата внутри Telegram.

## Целевая аудитория
- **Платит — работодатель:** кафе, бары, рестораны, кофейни (B2B).
- **Бесплатно — соискатель:** студенты, самозанятые, линейный персонал.

## Монетизация (гибрид)
| Источник | Цена | Канал |
|---|---|---|
| Подписка Pro / Business | 1 990 / 4 990 ₽/мес | ЮKassa |
| Boost вакансии в топ | микро | Telegram Stars |
| Супер-лайки «Срочно» | пакеты | Telegram Stars |
| Верификация компании (DaData) | разово | ЮKassa |

## Юнит-экономика (иллюстрация, пилот 1 город, ~150 активных заведений)
- Подписки: ~58 тыс ₽/мес; микротранзакции: ~6–9 тыс ₽/мес.
- **Итого регулярно ~65–70 тыс ₽/мес**; верификация ~87 тыс ₽ разово на старте.
- Рост до 3–5 городов (~1 000 заведений): **~400–600 тыс ₽/мес** регулярно.
- Драйверы: удержание (churn), конверсия, CAC — проверяются на пилоте.

## Технологии
- **Frontend:** Telegram Mini App (React + Vite + @telegram-apps/sdk).
- **Backend:** FastAPI + PostgreSQL/PostGIS, бот aiogram, S3 (Yandex Object Storage).
- **РФ-готовность:** ЮKassa, DaData, согласие 18+/152-ФЗ, хостинг в РФ.

## Статус
Рабочий продукт: вход через Telegram, свайпы, мэтчи, чат, смены, PDF-акт,
монетизация (Stars+ЮKassa с вебхуком), рефералы, отзывы/рейтинг, уведомления,
аналитика воронки, фильтры, загрузка фото. Backend покрыт тестами, CI на 3
компонента зелёный. Осталось: реальные ключи + пилот.
